== Description ==

Hungry Flamingo WP FSE Base Theme is a simple WordPress Block Theme supporting Full Site Editing. This theme is built for simplicity and lightspeed performance. Supporting responsive minimal design, global styles (since WP 6.0), and featuring a light and dark mode scheme.
